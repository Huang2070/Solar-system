# 阿尔兹海默症研究指南（2025 版）

## 执行摘要

阿尔兹海默病（Alzheimer's Disease, AD）研究在 2024-2025 年取得重大突破，特别是在**早期诊断**、**生物标志物**和**疾病修饰治疗**领域。本指南基于最新文献，为临床医生提供系统性的研究指导和实践建议。

### 核心进展概览

1. **诊断标准革新**：2024 年 NIA-AA 新标准将 AD 重新定义为生物学连续体，不再仅依赖临床症状 [1]
2. **血液检测突破**：血浆 p-tau217 检测准确性达 95-97%，可替代 PET 或脑脊液检测用于分诊 [2]
3. **治疗新时代**：抗 Aβ抗体药物（仑卡奈单抗等）获批，疾病修饰治疗成为现实 [3]
4. **精准医疗**：ATN 框架优化，实现个体化风险评估和干预时机选择 [4]

## 一、研究热点与前沿方向

### 1.1 优先推荐的研究方向

根据 2024-2025 年文献分析，以下方向具有高研究价值和临床转化潜力：

#### ⭐ 方向一：血液生物标志物的临床应用验证

**研究价值**：
- 血浆 p-tau217、p-tau231 在记忆门诊的实用性验证数据有限
- 不同人群（种族、合并症）的参考范围尚未建立
- 成本效益分析数据缺乏

**推荐研究设计**：

```
研究类型：前瞻性队列研究
样本量：500-1000 例（记忆门诊连续入组）

检测指标：
  - 血浆 p-tau217、p-tau181、Aβ42/40
  - 血清 GFAP、NfL

金标准对照：
  - Aβ-PET（阳性/negative）
  - CSF Aβ42、p-tau
  - 认知评估（MMSE、MoCA、CDR）

统计分析：
  - ROC 曲线下面积（AUC）比较
  - 敏感性、特异性、阳性/阴性预测值
  - 与 PET/CSF的一致性分析（Kappa 值）

创新点：
  - 建立中国人群参考范围
  - 评估合并症（高血压、糖尿病）对标志物的影响
  - 卫生经济学评价
```

**预期产出**：
- 发表 SCI 论文 1-2 篇（IF 5-10 分）
- 建立本地实验室参考区间
- 形成临床路径优化方案

---

#### ⭐ 方向二：临床前 AD 的早期识别与干预

**研究背景**：
- 2024 诊断标准强调临床前期识别的重要性
- p-tau231 对临床前 Aβ变化敏感 [2]
- 疾病修饰治疗窗口前移

**推荐研究设计**：

```
研究类型：纵向观察研究 + 随机对照试验（嵌套）

研究对象：
  - 纳入：60-75 岁，主观认知下降（SCD），CDR=0
  - 排除：客观认知损害、神经系统疾病

基线评估：
  - 血液生物标志物（p-tau217、Aβ42/40、GFAP）
  - Aβ-PET（可选）
  - 神经心理学测试（详细认知域评估）
  - MRI（海马体积、皮层厚度）
  - APOE ε4 基因型

随访计划：
  - 每 6 个月：认知评估、血液标志物
  - 每 12 个月：MRI、详细神经心理测试
  - 终点：转化为 MCI 或 AD 痴呆

亚组干预研究（可选）：
  - 高风险组（Aβ阳性）：生活方式干预 vs 常规护理
  - 干预内容：地中海饮食、有氧运动、认知训练
```

**创新点**：
- 聚焦超早期阶段（临床前）
- 多模态标志物联合预测模型
- 探索非药物干预效果

---

#### ⭐ 方向三：数字表型技术在 AD 筛查中的应用

**技术前沿**：
- 可穿戴设备监测步态、睡眠、活动模式
- 语音分析检测语言特征改变
- 眼动追踪识别视觉注意缺陷
- AI 算法整合多源数据（准确率 88-94%）[2]

**推荐研究设计**：

```
研究类型：诊断准确性研究 + 技术开发

研究对象：
  - AD 患者组（n=100）
  - MCI 组（n=150）
  - 健康对照组（n=200）

数据采集：
  1. 智能手表（7 天连续佩戴）：
     - 步态参数（步速、步幅变异性）
     - 睡眠质量（深睡比例、觉醒次数）
     - 日常活动模式
     
  2. 语音任务（标准化脚本）：
     - 图片描述任务
     - 自传体叙事
     - 语音特征：语速、停顿、词汇多样性
     
  3. 眼动追踪：
     - 视觉搜索任务
     - 面孔识别任务
     - 扫视/注视参数

AI 模型开发：
  - 机器学习算法（随机森林、XGBoost、深度学习）
  - 特征选择：LASSO 回归
  - 交叉验证：10 折
  - 外部验证：独立队列

输出指标：
  - 区分 AD vs 对照的 AUC
  - 预测 MCI 转化的风险评分
```

**优势**：
- 无创、低成本、可大规模筛查
- 适合基层医疗机构推广
- 符合数字医疗发展趋势

## 二、研究方法学指导

### 2.1 研究设计类型选择

#### 观察性研究（推荐初级研究者）

**横断面研究**
- **用途**：患病率调查、诊断准确性、相关性分析
- **优点**：周期短（6-12 月）、成本低、易实施
- **局限**：无法推断因果关系
- **示例**：《某地区 65 岁以上人群 AD 患病率及危险因素调查》

**队列研究**
- **用途**：发病率、危险因素、预后因素、疾病自然史
- **优点**：因果推断能力强、可计算发病率
- **局限**：随访时间长、失访偏倚、成本高
- **示例**：《社区老年人认知衰退轨迹及预测因素：5 年随访研究》

**病例对照研究**
- **用途**：罕见病病因探索、遗传因素研究
- **优点**：样本量小、周期短、适合罕见暴露
- **局限**：回忆偏倚、选择偏倚
- **示例**：《APOE ε4 基因型与早发性 AD 的关联研究》

---

#### 实验性研究（推荐有经验团队）

**随机对照试验（RCT）**
- **用途**：干预效果评价（药物、非药物）
- **设计要点**：
  1. 随机化方法：计算机生成随机序列、区组随机化
  2. 分配隐藏：密封信封、中央随机系统
  3. 盲法：双盲（受试者 + 研究者）、单盲、开放标签
  4. 对照选择：安慰剂对照、阳性对照、常规护理对照
  5. 样本量计算：基于主要终点、预期效应量、α=0.05、β=0.20（效能 80%）
  
- **注册要求**：必须在 ChiCTR 或 ClinicalTrials.gov 注册
- **报告规范**：遵循 CONSORT 声明

**实用性临床试验（PCT）**
- **特点**：真实世界环境、宽纳入标准、以患者为中心
- **优势**：外部真实性高、直接指导临床决策
- **示例**：《多奈哌齐联合美金刚 vs 单药治疗中重度 AD 的 PCT》

### 2.2 样本量计算方法

#### 两组均数比较（连续性变量）

**公式**：

```
n = 2 × [(Zα/2 + Zβ) × σ / δ]²

参数说明：
- n：每组样本量
- Zα/2：α错误对应的 Z 值（α=0.05 时，Z=1.96）
- Zβ：β错误对应的 Z 值（β=0.20 时，Z=0.84）
- σ：总体标准差（参考文献或预实验）
- δ：临床最小有意义差异
```

**实例**：

```
研究问题：新型认知训练 vs 常规护理对 MMSE 评分的影响

参数设定：
  - α = 0.05（双侧检验）
  - β = 0.20（检验效能 80%）
  - σ = 4.0（参考文献：MMSE 标准差约 4 分）
  - δ = 2.0（认为 MMSE 提高 2 分有临床意义）

计算：
n = 2 × [(1.96 + 0.84) × 4.0 / 2.0]²
n = 2 × [2.8 × 2]²
n = 2 × 31.36
n ≈ 63 例/组

考虑 20% 脱落率：
最终样本量 = 63 / 0.8 ≈ 79 例/组，总计 158 例
```

**在线工具推荐**：
- G*Power（免费软件，功能强大）
- Sample Size Calculator（ClinCalc.com）
- PASS（商业软件，功能全面）

---

#### 两组率比较（分类变量）

**公式**：

```
n = [Zα/2√(2p̄q̄) + Zβ√(p₁q₁ + p₂q₂)]² / (p₁ - p₂)²

参数说明：
- p₁、p₂：两组预期事件发生率
- q₁ = 1-p₁，q₂ = 1-p₂
- p̄ = (p₁ + p₂) / 2
- q̄ = 1 - p̄
```

**实例**：

```
研究问题：Aβ抗体治疗 vs 安慰剂延缓疾病进展的效果

参数设定：
  - 对照组进展率 p₁ = 0.40（40%）
  - 治疗组进展率 p₂ = 0.25（期望降低至 25%）
  - α = 0.05，β = 0.20

计算：
p̄ = (0.40 + 0.25) / 2 = 0.325
q̄ = 1 - 0.325 = 0.675

n = [1.96×√(2×0.325×0.675) + 0.84×√(0.40×0.60 + 0.25×0.75)]² / (0.40 - 0.25)²
n = [1.96×0.663 + 0.84×0.654]² / 0.0225
n = [1.30 + 0.55]² / 0.0225
n = 3.42 / 0.0225
n ≈ 152 例/组

考虑 15% 脱落率：
最终样本量 = 152 / 0.85 ≈ 179 例/组，总计 358 例
```

### 2.3 研究对象选择标准

#### 纳入标准模板（根据研究类型调整）

**AD 痴呆组**：

```
1. 年龄 50-85 岁
2. 符合 NIA-AA 2024 年 AD 诊断标准 [1]
3. CDR 总分 ≥ 0.5，且 CDR-SOB ≥ 4
4. MMSE 评分 10-26 分
5. MoCA 评分 < 26 分
6. 至少一项 AD 生物标志物阳性：
   - Aβ-PET 阳性（Centiloid > 20）
   - CSF Aβ42 < 1000 pg/mL 或 Aβ42/40 < 0.07
   - 血浆 p-tau217 > 阈值（根据实验室参考值）
7. 有可靠的知情者（可提供病史）
8. 签署知情同意书
```

**MCI 组**：

```
1. 年龄 55-80 岁
2. 主诉认知下降（本人或知情者报告）
3. 客观认知损害证据：
   - 至少一个认知域低于常模 1-1.5 SD
   - 但日常生活能力基本保留（FAQ ≤ 9）
4. CDR = 0.5
5. 不符合痴呆诊断标准（DSM-5）
6. 稳定用药至少 3 个月（如适用）
```

**健康对照组**：

```
1. 年龄性别匹配（±3 岁）
2. 主观认知功能正常
3. 客观认知测试正常：
   - MMSE ≥ 27 分（受教育年限校正）
   - MoCA ≥ 26 分
4. CDR = 0
5. 无神经系统疾病史
6. 汉密尔顿抑郁量表（HAMD）< 7 分
```

#### 排除标准（通用）

```
1. 其他类型痴呆（血管性、路易体、额颞叶等）
2. 严重精神疾病（精神分裂症、双相障碍、重度抑郁）
3. 酒精或药物滥用史
4. 严重躯体疾病（心肝肾功能不全、恶性肿瘤晚期）
5. 甲状腺功能异常（TSH 超出正常范围）
6. 维生素 B12 或叶酸缺乏
7. 梅毒或 HIV 感染
8. 头部外伤史（昏迷 > 30 分钟）
9. 癫痫发作史
10. MRI 禁忌证（幽闭恐惧、金属植入物等）
11. 参与其他干预性临床试验（30 天内）
```

### 2.4 结局指标选择

#### 主要终点（Primary Endpoint）

**认知功能指标**：

| 量表名称 | 评估域 | 评分范围 | 临床意义 | 推荐场景 |
|---------|-------|---------|---------|---------|
| MMSE | 整体认知 | 0-30 | <24 提示损害 | 筛查、轻度损害 |
| MoCA | 多认知域 | 0-30 | <26 提示损害 | MCI、早期 AD |
| ADAS-Cog | 认知功能 | 0-70 | 分数越高损害越重 | 药物试验金标准 |
| CDR-SOB | 临床痴呆 | 0-18 | ≥4 提示 AD | 分期、疗效评价 |

**生物标志物指标**：
- **Aβ相关**：PET Centiloid 值、CSF Aβ42、血浆 Aβ42/40
- **Tau 相关**：CSF p-tau181、血浆 p-tau217
- **神经变性**：MRI 海马体积、FDG-PET 代谢、CSF NfL
- **新兴标志物**：血清 GFAP、炎症因子（IL-6、TNF-α）

**功能指标**：
- ADCS-ADL（日常生活能力）
- FAQ（功能活动问卷）
- QoL-AD（生活质量）

### 3.2 神经心理学评估流程

**标准化评估顺序**（总耗时约 90-120 分钟）：

```
第一阶段：筛查测试（15 分钟）
  1. MMSE（10 分钟）
  2. CDR 评估（5 分钟，需知情者配合）

第二阶段：详细认知评估（60 分钟）
  3. MoCA（15 分钟）
  4. 听觉词语学习测验（AVLT）（15 分钟）
    - 即刻回忆（5 次学习）
    - 短延迟回忆
    - 长延迟回忆
    - 再认
  5. 数字广度测验（10 分钟）
    - 顺背数字
    - 倒背数字
  6. 画钟测验（CDT）（5 分钟）
  7. 言语流畅性测验（10 分钟）
    - 动物命名（1 分钟）
    - 蔬菜命名（1 分钟）
    - 词语流畅性（F/A/S 各 1 分钟）
  8. 连线测验（TMT）（10 分钟）
    - TMT-A（数字连接）
    - TMT-B（数字字母转换）

第三阶段：情绪与行为评估（15 分钟）
  9. GDS-15（老年抑郁量表）（10 分钟）
  10. NPI-Q（神经精神症状）（5 分钟，知情者报告）

第四阶段：功能评估（15 分钟，需知情者）
  11. FAQ（功能活动问卷）（10 分钟）
  12. ADCS-ADL（20 分钟）
```

**评估质量控制**：
- 评估员需经过统一培训（一致性 Kappa > 0.8）
- 固定评估时间（上午 9-11 点或下午 3-5 点）
- 安静独立房间，避免干扰
- 使用标准化指导语（录音备份）
- 原始记录保存（纸质 + 电子双录入）

### 3.3 生物样本采集 SOP

#### 静脉血采集（空腹）

```
准备材料：
  - EDTA 抗凝管（紫色帽）× 2
  - 促凝管（红色帽）× 1
  - 冰盒（预冷至 4℃）
  - 离心机（预冷至 4℃）
  - 冻存管（0.5 mL 或 1.5 mL）
  - 标签打印机

采集流程：
  1. 受试者空腹 8-12 小时（可饮水）
  2. 上午 8:00-10:00 采集（减少昼夜节律影响）
  3. 肘静脉穿刺，弃去前 2 mL 血
  4. 按顺序采血：
     a. EDTA 管 5 mL（用于血浆分离）
     b. 促凝管 5 mL（用于血清分离）
     c. EDTA 管 2 mL（用于 DNA 提取）
  5. 轻柔颠倒混匀 5-8 次（勿剧烈震荡）

处理流程：
  1. 标本立即置于冰盒（4℃）
  2. 30 分钟内离心：
     - 血浆：1500 × g，15 分钟，4℃
     - 血清：室温凝固 30 分钟后，2000 × g，10 分钟
  3. 分装：
     - 血浆/血清分装至冻存管（每管 0.5 mL）
     - 避免反复冻融（分装足够单次检测用量）
  4. 标记：
     - 研究编号_样本类型_日期（例：AD001_PLA_20250403）
  5. 储存：
     - -80℃超低温冰箱（长期保存）
     - 避免放在冰箱门侧（温度波动大）
     - 建立样本库存管理系统
```

#### 脑脊液采集（腰椎穿刺）

```
适应证：
  - 疑似 AD 但血液标志物不明确
  - 需要鉴别诊断（感染、炎症、副肿瘤）
  - 科研需要（生物标志物研究）

禁忌证：
  - 颅内压增高（视乳头水肿）
  - 穿刺部位感染
  - 凝血功能障碍（INR > 1.5，血小板 < 50×10⁹/L）
  - 抗凝治疗（华法林停药 5 天，DOAC 停药 48 小时）

操作流程：
  1. 术前评估：
     - 签署知情同意书
     - 血常规、凝血功能检查
     - 头颅 CT/MRI（排除占位）
  2. 体位：左侧卧位，屈颈抱膝
  3. 穿刺点：L3-L4 或 L4-L5 间隙
  4. 消毒铺巾，利多卡因局麻
  5. 穿刺针进入蛛网膜下腔，测开口压（正常 80-180 mmH₂O）
  6. 留取 CSF：
     - 第 1 管 2 mL（生化、常规）
     - 第 2 管 5 mL（Aβ42、p-tau）
     - 第 3 管 2 mL（备用）
  7. 术后平卧 4-6 小时（预防低压性头痛）

CSF 处理：
  1. 立即离心：2000 × g，10 分钟，4℃
  2. 上清液分装（每管 0.5 mL）
  3. -80℃保存（避免反复冻融）
  4. 检测指标：
     - Aβ42、Aβ40（ELISA 或化学发光）
     - Total-tau、p-tau181
     - NfL（神经丝轻链）
```

### 3.4 影像学检查 protocol

#### 头颅 MRI（3.0T 推荐）

```
扫描序列（总耗时约 45 分钟）：

1. 3D-T1 加权像（结构像）：
   - 序列：MPRAGE 或 SPGR
   - TR/TE：2300/2.98 ms
   - 翻转角：9°
   - 层厚：1.0 mm（无间隔）
   - FOV：256 × 256 mm
   - 矩阵：256 × 256
   - 用途：体积测量（海马、全脑）、皮层厚度分析

2. T2 加权像：
   - TR/TE：4000/100 ms
   - 层厚：5 mm
   - 用途：病灶筛查（梗死、肿瘤、脱髓鞘）

3. FLAIR 序列：
   - TR/TE/TI：9000/100/2500 ms
   - 层厚：5 mm
   - 用途：白质高信号（WMH）评估

4. DWI（弥散加权成像）：
   - b 值：0, 1000 s/mm²
   - 用途：急性梗死、朊病毒病

5. SWI（磁敏感加权成像）：
   - 用途：微出血、铁沉积

6. DTI（弥散张量成像，可选）：
   - 方向数：30 方向
   - b 值：1000 s/mm²
   - 用途：白质纤维束完整性

后处理分析：
  - FreeSurfer：自动分割（海马体积、皮层厚度）
  - SPM：体素形态学分析（VBM）
  - FSL：DTI 分析（FA 值、MD 值）
  - 人工质控：由放射科医师审核
```

#### Aβ-PET / Tau-PET（有条件推荐）

```
示踪剂选择：
  - Aβ-PET：18F-florbetapir、18F-flutemetamol、18F-florbetaben
  - Tau-PET：18F-flortaucipir、18F-MK-6240

扫描流程：
  1. 注射示踪剂：静脉注射 18F 标记示踪剂（约 370 MBq）
  2. 等待分布：静卧 40-60 分钟（避光、少交流）
  3. PET 扫描：20-30 分钟（3D 模式）
  4. 衰减校正：CT 或 68Ge 源

图像分析：
  - 目测法：阳性/阴性（由认证阅片师判读）
  - 半定量：SUVr（标准化摄取值比值）
    - 参考区：小脑（Aβ-PET）、枕叶皮质（Tau-PET）
    - 靶区：额叶、顶叶、颞叶、扣带回
    - 阳性阈值：SUVr > 1.10（Aβ）、SUVr > 1.25（Tau）
  - Centiloid 标度：0-100（0=年轻健康，100=典型 AD）
    - 阳性阈值：Centiloid > 20
```

## 四、统计分析方法

### 4.1 描述性统计

```
连续变量：
  - 正态分布：均数 ± 标准差（Mean ± SD）
  - 偏态分布：中位数（四分位距）[M（P25, P75）]
  - 组间比较：t 检验（正态）或 Mann-Whitney U 检验（偏态）

分类变量：
  - 频数（百分比）[n（%）]
  - 组间比较：χ²检验或 Fisher 精确检验

多组比较：
  - 方差分析（ANOVA）+ 事后检验（LSD 或 Bonferroni）
  - 或非参数 Kruskal-Wallis H 检验
```

---

### 4.2 诊断准确性分析

```
ROC 曲线分析：
  - 计算 AUC（曲线下面积）
  - 最佳截断值：Youden 指数最大化
  - 敏感性、特异性、阳性预测值、阴性预测值
  - 95% 置信区间

示例代码（R 语言）：

library(pROC)
roc_obj <- roc(group, biomarker_level)
auc(roc_obj)  # AUC 值
ci.auc(roc_obj)  # 95% CI
coords(roc_obj, "best", ret=c("threshold", "sensitivity", "specificity"))

一致性分析：
  - Kappa 统计量（血液标志物 vs PET/CSF）
  - Kappa > 0.75：优秀一致性
  - 0.4 < Kappa < 0.75：中等一致性
  - Kappa < 0.4：较差一致性
```

---

### 4.3 生存分析（纵向研究）

```
Kaplan-Meier 曲线：
  - 事件：MCI 转化为 AD 痴呆
  - X 轴：随访时间（月）
  - Y 轴：无事件生存率
  - 组间比较：Log-rank 检验

Cox 比例风险回归：
  - 单因素分析：筛选潜在预测因子
  - 多因素分析：调整混杂因素
  - 输出：风险比（HR）及 95% CI

示例代码（R 语言）：

library(survival)
cox_model <- coxph(Surv(time, event) ~ age + sex + APOE4 + ptau217, data=mydata)
summary(cox_model)
forestplot(cox_model)  # 森林图
```

---

### 4.4 多变量回归分析

```
线性回归（连续结局）：
  - 结局变量：MMSE 变化值、海马萎缩率
  - 自变量：年龄、性别、教育、生物标志物
  - 假设检验：共线性（VIF < 5）、残差正态性、同方差性

Logistic 回归（二分类结局）：
  - 结局：Aβ-PET 阳性/阴性、快速进展者/缓慢进展者
  - 输出：比值比（OR）及 95% CI

示例代码（R 语言）：

# 线性回归
lm_model <- lm(MMSE_change ~ age + sex + education + baseline_ptau, data=mydata)
summary(lm_model)

# Logistic 回归
glm_model <- glm(PET_positive ~ age + APOE4 + ptau217, family=binomial, data=mydata)
exp(cbind(OR = coef(glm_model), confint(glm_model)))  # OR 及 95% CI
```

---

### 4.5 中介效应与调节效应分析

```
中介效应（Mediation Analysis）：
  - 研究问题：生物标志物是否通过脑萎缩介导认知下降？
  - 路径：X（Aβ沉积）→ M（海马萎缩）→ Y（认知下降）
  - 方法：Bootstrap 法（重复抽样 5000 次）
  - 软件：PROCESS macro for SPSS 或 R 包 'mediation'

调节效应（Moderation Analysis）：
  - 研究问题：APOE ε4 是否调节 Aβ对认知的影响？
  - 模型：Y = β₀ + β₁X + β₂M + β₃(X×M) + ε
  - 交互项显著性：β₃ ≠ 0 表示存在调节效应
```

### 7.2 标书撰写要点

**立项依据（4000 字以内）**

```
逻辑链条：
1. 疾病负担（AD 患病率、经济负担、社会影响）
2. 临床痛点（早期诊断困难、缺乏有效治疗）
3. 研究现状（国内外进展、主流观点）
4. 存在问题（知识空白、争议点）
5. 科学假说（提出你的创新思路）
6. 研究意义（理论价值 + 临床应用）

写作技巧：
- 引用权威文献（Nature、Lancet 等高分文章）
- 图表辅助（技术路线图、机制假说图）
- 突出紧迫性和必要性
```

**研究内容与技术路线**

```
研究内容（分 3-4 个部分）：
  1. 横断面研究：血液标志物诊断准确性
  2. 纵向研究：预测疾病进展
  3. 机制探索：与影像学、遗传学关联
  4. （可选）干预研究：标志物指导的个体化治疗

技术路线：
  - 流程图形式呈现
  - 清晰展示各环节逻辑关系
  - 标注关键节点和时间进度
```

**可行性分析**

```
✓ 研究基础：已发表相关论文 X 篇，预实验数据显示...
✓ 平台条件：拥有 3.0T MRI、PET-CT、SPF 级动物房
✓ 样本资源：生物样本库存量 XXX 份，随访队列 XXX 人
✓ 团队配置：高级职称 X 人，中级 X 人，研究生 X 人
✓ 合作网络：与 XX 医院、XX 研究所建立合作
```


## 十、结语与建议

### 给初入 AD 研究领域医生的建议

**起步阶段（第 1-2 年）**：
1. **从临床问题出发**：不要盲目追求热点，思考你在门诊遇到的实际困惑
2. **加入成熟团队**：向有经验的研究组学习，参与多中心项目积累经验
3. **掌握核心技能**：统计学基础、文献检索、论文写作
4. **小规模试点**：先做横断面研究或病例系列，熟悉研究全流程
5. **建立合作关系**：与检验科、放射科、病理科建立良好协作

**发展阶段（第 3-5 年）**：
1. **确定研究方向**：聚焦 1-2 个细分领域，形成特色
2. **申请基金项目**：从院级、市级逐步到省级、国家级
3. **培养研究生**：组建自己的小团队
4. **学术发声**：参加国内外会议，建立学术网络
5. **持续学习**：关注最新进展（AAIC、AD/PD 会议）

**成熟阶段（5 年以上）**：
1. **引领方向**：提出原创性科学假说
2. **多中心合作**：发起全国性或国际性研究
3. **指南制定**：参与行业共识和指南编写
4. **转化应用**：推动研究成果进入临床实践
5. **人才培养**：培养下一代研究者

```r
# 参数：d=效应量 (Cohen's d), power=检验效能，sig.level=显著性水平
pwr.t.test(d=0.5, power=0.8, sig.level=0.05, type="two.sample")
# 输出：每组约 64 例

library(powerSurvEpi)
```
