# 成员内部类的写法

```Java
//在外部类里定义
@Data
public class OuterClass {
    private String a;
    private String b;

    @Data
    public class InnerClass {
        private String c;
        private String d;
    }
}

//new成员类的方式
OuterClass aaa = new OuterClass();
OuterClass.InnerClass bbb = aaa.new InnerClass();
```
