# 定义接口
```Java
public interface Config {
    //执行逻辑
    String excute();

    //执行类型
    String getType();

    //通用方法
    void commonFunc();
}
```


# 定义抽象类,继承接口
```Java
public abstract class AbstractConfig implements Config {

    //执行逻辑
    String excute() {
        String type = getType();
        //执行逻辑,根据type执行不同逻辑

    }

    //通用方法,可以在子类里被调用
    void commonFunc() {
        //逻辑
    }
}
```

# 定义实现类,继承抽象类
```Java
public abstract class Config1 extends AbstractConfig {

    //定义执行类型
    String getType() {
        return "config1";
    }

    //在子类里可以执行通用的方法
    public commonFunc() {
        super.commonFunc();
    }

}

public abstract class Config2 extends AbstractConfig {
    ...
}
```

# 使用
```Java
main() {
    //定义子类
    Config1 config1 = new Config1();

    //子类可以执行抽象父类的方法. 因为子类自动继承了父类继承的接口里的方法. 用这种方式可以让子类执行父类的方法.
    config1.commonFunc();

    config1.execute();
}
```
